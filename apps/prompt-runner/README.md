# @axm/prompt-runner

Codex prompt runner CLI for this monorepo.

## Usage

```bash
bun run apps/prompt-runner/src/cli.ts checklist packages/zod-helpers
bun run apps/prompt-runner/src/cli.ts llms packages/zod-helpers
bun run apps/prompt-runner/src/cli.ts typedoc packages/zod-helpers
```

## Docs

Generated documentation lives in `docs/` and can be updated with:

```bash
bun run docs
```
